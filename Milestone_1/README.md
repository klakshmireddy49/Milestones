
# Milestone 1: Environment Setup & Basic Agent Creation

## Objective
Install LangChain and create a foundational conversational agent capable of handling basic user queries.

## Tasks
- Set up Python and LangChain development environment  
- Explore LangChain core components:
  - LLMs (Large Language Models)
  - Prompts
  - Chains
  - Agents  
- Connect to a language model (real or simulated)  
- Build a simple LangChain agent to respond to static queries  
- Experiment with agent types (e.g., zero-shot-react-description)  
- Create prompt templates and implement basic input/output logic  
- Develop a console interface for interactive testing  

## Output
- Functional LangChain environment  
- Basic single-agent prototype with prompt-based interaction  
- Console-based interface for user interaction  

## Evaluation Criteria
- LangChain agent successfully built  
- Agent responds correctly to user prompts
